# 🚀 Thapar EduTube MVP - Quick Start (5 minutes)

## ⚡ Super Quick Start

### What is This?
A **TikTok-style educational video player** for Thapar University. Swipe up/down to watch videos, swipe left/right to change topics.

### Prerequisites
- Flutter 3.24+
- Android 8.0+ or iOS 13.0+
- Stable internet connection

---

## 🎯 Run It (2 minutes)

```bash
# Navigate to project
cd edutube_shorts

# Get dependencies
flutter pub get

# Run on device/emulator
flutter run
```

**That's it!** The app should now be running.

---

## 🎮 Try It (3 minutes)

### What You'll See
```
┌─────────────────────────┐
│ Thapar EduTube          │
├─────────────────────────┤
│ • Numerical Analysis    │
│ • Data Structures       │
│ • Algorithms            │
└─────────────────────────┘
```

### What to Do
1. **Tap** "Numerical Analysis"
2. **Swipe UP** → Next video
3. **Swipe DOWN** → Previous video
4. **Swipe LEFT** → Next topic
5. **Swipe RIGHT** → Previous topic
6. **Tap video** → Play/Pause

## 🐛 Something Wrong?

### App Won't Run
```bash
flutter clean
flutter pub get
flutter run
```

### Video Won't Play
- Check internet connection
- Try different video (swipe up/down)
- Check video_player package installed

### Gesture Not Working
- Try swiping from different position
- For horizontal swipe, must be at topic boundary
- Check screen orientation
